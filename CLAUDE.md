# CLAUDE.md
## Project: Premium Speedboat Booking Landing Page (Punnamada, Alleppey)

---

## 🎯 Objective

Build a **single-page, SEO-optimized, high-conversion landing page** for a speedboat business in **Punnamada, Alleppey**.

Primary goal:
- Maximize **lead conversions (calls / WhatsApp / bookings)**
- Rank for **local commercial-intent keywords**
- Deliver a **premium, trust-heavy UI**

---

## 🧠 Positioning Strategy

This is NOT a generic tourism page.

We are selling:
- Speed
- Experience
- Exclusivity
- Convenience

Target users:
- Tourists searching *"speed boat alleppey"*
- Couples / small groups
- Last-minute decision makers

---

## ⚠️ Critical Constraints

- Single page only (no routing complexity)
- Must load fast (<2.5s LCP target)
- Mobile-first (80% traffic expected)
- No clutter, no “cheap tourism look”

---

## 🧩 Tech Stack

- Next.js (App Router)
- Tailwind CSS
- UI Libraries:
  - shadcn/ui → core components
  - Magic UI → subtle animations
  - Aceternity UI → premium visual sections

Avoid:
- Heavy animation frameworks
- Overuse of gradients
- Anything that slows down TTI

---

## 🎨 Design Language

### Principles
- Clean, minimal, high-end
- Strong typography hierarchy
- Spacious layout (padding > content density)
- Sharp edges or soft radius (consistent)

### Colors
- Base: White / Off-white
- Accent: Deep blue / teal (water association)
- Text: Near-black (not pure black)

### Typography
- Headings: Bold, confident
- Body: Clean, highly readable
- Avoid decorative fonts

---

## 🧱 Page Structure (STRICT)

### 1. Hero Section (Above the Fold)
Purpose: Convert instantly

Content:
- Headline:
  "Premium Speed Boat Ride in Alleppey – Punnamada Lake"
- Subtext:
  Short, benefit-driven (not generic tourism fluff)
- CTA:
  - "Book Now"
  - "Call Now"
- Visual:
  Real speedboat image (NOT stock-looking)

Must include:
- Location mention (SEO signal)
- Trust trigger (e.g., "1000+ rides completed")

---

### 2. Quick Highlights Bar
Purpose: Fast scanning

Items:
- Duration (e.g., 30 min / 1 hr)
- Capacity
- Pricing starting point
- Location: Punnamada

---

### 3. Experience Section
Purpose: Sell the feeling

Content:
- Short paragraphs
- Bullet benefits:
  - High-speed ride
  - Scenic backwaters
  - Private experience
  - Flexible timing

Avoid:
- Generic “beautiful nature” filler

---

### 4. Pricing Section
Purpose: Reduce friction

Include:
- Clear packages
- No hidden pricing feel
- CTA under each package

Optional:
- “Most popular” highlight

---

### 5. Gallery Section
Purpose: Visual proof

Requirements:
- Real images only
- Mix:
  - Boat shots
  - Customer shots
  - Water views

Grid layout:
- Masonry or clean grid
- No carousel overload

---

### 6. Why Choose Us
Purpose: Trust building

Items:
- Local operator (not reseller)
- Safety ensured
- Transparent pricing
- Easy booking

---

### 7. Testimonials (Optional but recommended)
Purpose: Social proof

- Keep it short
- Avoid fake-looking content

---

### 8. Location Section
Purpose: Local SEO boost

Content:
- Map embed (Punnamada Lake)
- Short text:
  "Located in Punnamada, Alleppey..."

Include keywords naturally.

---

### 9. FAQ Section
Purpose: SEO + objections

Questions:
- Best time for speedboat ride?
- Is it safe?
- How to book?
- Cancellation policy?

---

### 10. Final CTA Section
Purpose: Close conversion

- Strong CTA:
  "Book Your Speedboat Ride Now"
- Buttons:
  - Call
  - WhatsApp

## 🔍 SEO Strategy

### Keywords (Primary)
- speed boat alleppey
- alleppey speed boat ride
- punnamada speed boat

### Keywords (Secondary)
- boat ride alleppey
- backwater speed boat
- alleppey boating price

---

### On-page Rules

- H1 → Only ONE (Hero)
- H2 → Section headings
- Include location keywords naturally
- Avoid keyword stuffing

---

### Technical SEO

- Use semantic HTML
- Add:
  - meta title
  - meta description
- OpenGraph tags
- Structured data:
  - LocalBusiness schema

---

## ⚡ Performance Rules

- Use next/image for all images
- Lazy load below-the-fold images
- Compress images (WebP preferred)
- Avoid large JS bundles

---

## 🔘 Conversion Optimization

Must include:
- Sticky mobile CTA (Call / WhatsApp)
- Click-to-call enabled
- Minimal form (if used)

Remove:
- Any unnecessary input fields
- Distractions

---

## 🚫 Common Mistakes (DO NOT DO)

- Over-designing (kills trust)
- Too many animations
- Generic tourism copy
- Fake testimonials
- Slow loading images
- No clear CTA above the fold

---

## 🧠 UX Heuristics

- User should understand offering in <5 seconds
- CTA always visible within one scroll
- No cognitive overload

---

## 🏁 Definition of Done

- Loads fast on 4G
- Mobile UX feels smooth
- Clear conversion path
- Ranks for local keywords over time
- Looks premium, not templated
